# OTFcoder-Practical
Practical Tast For OTFcoder
## How to Install
- cd projectname
- composer install
- php artisan key:generate
- Create a database and inform .env
- php artisan migrate --seed to create and      populate tables
- php artisan serve to start the app on http://localhost:8000/
